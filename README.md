# Programa de Formação em Elixir | Teste Técnico

## Como executar a solução

- Abrir o DesafioStone.sln com o visual studio
- Executar o programa e ver o resultado no console


## Apontamentos

Utilizei as estruturas de dados List e Dictionary do C# mockando os dados da lista de compra e dos emails. Com as duas listas em mãos, realizei os cálculos para chegar ao valor final dos produtos e dividir entre as pessoas.
Feito isso, populei o dicionário utilizando a lista de emails e distribuindo os valores entre elas. Separei em três classes diferentes (Item, Program e Desafio), me utilizando do paradigma de Orientação a Objetos.
